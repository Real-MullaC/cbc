- ```Ctrl + C```: Copy
- ```Ctrl + V```: Paste
- ```Ctrl + X```: Cut
- ```Ctrl + P```: Print
- ```Ctrl + F```: Find (Works on ProSolution)
- ```Ctrl + S```: Save
- ```Ctrl + Z```: Undo
- ```Ctrl + Y```: Redo
- ```Ctrl + C```: Copy
- ```Shift + F5```: Start Slideshow from Current Slide
- ```F5```: Start Slideshow from Start
- ```Alt + Tab```: Switch Windows Quickly
